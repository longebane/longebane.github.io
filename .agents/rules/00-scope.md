# Project Scope: longebane.github.io

## Mission

This is a personal portfolio and hobbyist website built with **Astro**. The goal is to provide a central hub for personal branding, music projects, and technical showcases.

## Boundaries

- **Content:** Primarily static content with some interactive components.
- **Visuals:** Must strictly adhere to the established "premium/personal" aesthetic.
- **Accessibility:** All UI elements must aim for high accessibility standards (WCAG).
- **Tooling:** All development lifecycle tasks must use the `vp` (Vite+) toolchain.
